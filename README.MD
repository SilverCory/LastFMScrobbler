# LastFMScrobbler
**A last FM scrobbler for discord using Rich Presence.**

## Building
You need go dep.

- GOPATH=$(pwd)
- cd src/github.com/SilverCory/LastFMScrobbler/
- dep ensure
- go build src/github.com/SilverCory/LastFMScrobbler/main.go

**TADA!**
